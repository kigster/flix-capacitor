FlixCapacitor
=============

A little flux/flix capacitor box based on Teensy 3.1 for showing pictures, clock, playing DJ sets, and starting a small rave in a recovery room.


Library dependencies:

```
 Time
 ILI9341_t3
 SPI
 SD
 Audio
 Wire
 Sparkfun7SD
 TeensyRTC
 SoftwareSerial
```
